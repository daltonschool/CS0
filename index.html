Instructions To Make Changes

  1. Open Terminal
  2. Copy and paste this command: 
    git clone https://github.com/daltonschool/CS0.git
  3. Hit enter.
  4. A folder named CS0 with bunch of stuff in it will download to your computer.
  5. Use Sublime Text to open the file(s) you want to change.
  6. Make changes and save.
  7. Back in Terminal copy and paste these instructions: 
    cd CS0
git add *
git commit -m "small changes"
git push
  8. Hit enter and you will have pushed your changes to the website.
  9. Refresh the webpage. Sometimes it take 5 or so mins to stick.
  10. From now on, every time you want to change something, go to Terminal and type 
    git pull
    then go to Step 5.
